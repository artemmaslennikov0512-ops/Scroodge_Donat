# Дорожная карта развития проекта

Список фич и улучшений для последующей реализации.

**Для приоритета MVP** см. [docs/MVP-RECOMMENDATIONS.md](docs/MVP-RECOMMENDATIONS.md) — выводы ревью, соответствие коду и чеклист срочных задач (API донатов/алертов, цели на странице стримера, webhook + Goal, Discord).

---

## 1. Мобильное приложение (PWA)
- Добавить манифест (`manifest.json`) и service worker (например, через next-pwa).
- Настроить офлайн-кеширование статики и API-запросов (ограниченно).

---

## 2. Расширенная аналитика и прогнозирование
- Собрать данные о донатах в отдельную таблицу `DonationAnalytics`.
- Использовать библиотеки для ML (например, tensorflow.js) или внешние сервисы для прогнозов.
- Создать дашборд с графиками и предсказаниями.

---

## 3. Автопилот для алертов
- Добавить поле `alertTheme` в настройки пользователя.
- По расписанию (или через AI-сервис, например, анализ чата) менять тему.

---

## 4. Криптовалюты и NFT
- Интеграция с платежными шлюзами: Coinbase Commerce, NowPayments.
- Выпуск NFT через API OpenSea или собственный смарт-контракт.

---

## 5. Подписки на стримеров (Patreon-подобная система)
- Таблицы: `CreatorSubscriptions`, `SubscriptionTiers`, `UserSubscriptions`.
- Ежемесячные списания через Stripe.

---

## 6. Мультиязычность
- Использовать next-i18next или встроенный next-intl.
- Перевести все тексты на несколько языков.

---

## 7. Интерактивные мини-игры
- Компоненты игр: колесо фортуны, голосование.
- WebSockets для синхронизации.

---

## 8. AI-модерация
- Интеграция с Google Perspective API или собственная модель.
- Автоматическая блокировка токсичных сообщений.

---

## 9. База знаний
- Статический сайт с документацией (можно отдельно на Next.js).
- Видео-гайды на YouTube с встраиванием.

---

## 10. Тикет-система (улучшенная)
- Категории, приоритеты, назначение ответственных.
- Интеграция Telegram-бота для уведомлений.

---

## 11. Автоматическая верификация через Госуслуги
- API ЕСИА для подтверждения личности.
- Хранить только статус верификации, не сами данные.

---

## 12. Партнерская программа
- Уже начата.

## 13. Система достижений и бейджей для донатеров
- Таблицы: `Achievement`, `UserAchievement`.
- Начисление за действия (сумма донатов, количество).

## 14. Розыгрыши и конкурсы
- Таблица `Giveaway` с призами и условиями.
- Автоматический выбор победителя по завершению.

## 15. Брендирование и white label
- Поле `branding` в настройках пользователя (логотип, цвета).

## 16. PWA
- См. пункт 1.

## 18. Кэширование
- Redis для сессий и частых запросов.

## 19. CI/CD

## 20. Мониторинг

## 22. Age verification
