# Дизайн: сайт донатов

По правилам из плана (на основе 1tips).

## Токены и палитра

- **Primary:** синий (rgb 59 130 246), hover светлее. Используется для основного CTA и фокуса.
- **Accent:** фиолетовый — зарезервирован для акцентов при расширении.
- **Текст:** контраст по WCAG AA (4.5:1 для обычного текста, 3:1 для крупного и UI).
- Токены заданы в `app/globals.css` через CSS-переменные.

## Доступность

- **focus-visible:** кнопки и поля имеют `focus:outline-none focus-visible:ring-2` (или `focus:ring-2`), чтобы фокус был виден при навигации с клавиатуры.
- **prefers-reduced-motion:** в `globals.css` анимации и переходы сокращаются до 0.01ms при запросе пользователя.
- Подписи полей связаны с полями через `htmlFor` / `id`.

## Компоненты

- DonationWidget: один главный CTA («Поддержать»), группировка полей, ошибки без утечки внутренних деталей.
- Цвета виджета и кнопок через токены (`--color-primary`, `--color-error`, `--color-border` и т.д.).

## Рекомендации

- Периодически проверять контраст пар «текст/фон» и «кнопка/фон» (например, WebAIM Contrast Checker).
- Не добавлять новые анимации без учёта `prefers-reduced-motion`.
