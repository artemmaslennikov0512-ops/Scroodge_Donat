# Публичный API для стриминговых интеграций

Используйте эти эндпоинты в OBS, StreamElements, StreamLabs и других сервисах для отображения донатов и статистики.

## Базовый URL

- Разработка: `http://localhost:3000`
- Продакшен: `https://your-domain.com`

## Эндпоинты

### 1. Последние донаты (для оверлея)

```
GET /api/v1/streamers/{slug}/donations
```

**Параметры (query):**

| Параметр | Тип   | По умолчанию | Описание        |
|----------|--------|---------------|------------------|
| `limit`  | number | 20            | Кол-во записей (макс. 100) |

**Пример ответа:**

```json
{
  "donations": [
    {
      "id": "clx...",
      "donorName": "Аноним",
      "amount": 500,
      "message": "Крутой стрим!",
      "createdAt": "2026-03-01T15:30:00.000Z"
    }
  ]
}
```

**Пример запроса (OBS / браузерный источник):**

```
https://your-domain.com/api/v1/streamers/stream_king/donations?limit=10
```

Обновляйте источник каждые 5–10 секунд.

---

### 2. Статистика стримера

```
GET /api/v1/streamers/{slug}/stats
```

**Пример ответа:**

```json
{
  "streamerId": "clx...",
  "displayName": "Stream King",
  "totalAmount": 125000,
  "totalDonations": 340,
  "donationsLast24h": 12
}
```

Используйте для виджетов «Всего донатов», «Собрано за сегодня» и т.п.

---

## Готовый оверлей для OBS (Browser Source)

Используйте страницу оверлея — она сама подгружает донаты и обновляется каждые 6 секунд:

```
https://your-domain.com/streamer/{slug}/overlay?transparent=1&limit=10
```

- `transparent=1` — прозрачный фон для наложения на стрим.
- `limit` — число последних донатов (1–20).

Пошаговая настройка для стримера: [STREAMER_SETUP.md](STREAMER_SETUP.md).

## Интеграция в OBS (свой виджет)

1. Добавьте источник **Браузер** (Browser Source).
2. Укажите URL готового оверлея (см. выше) или своей страницы-виджета.
3. Рекомендуемая ширина/высота: 400×400 (оверлей) или 400×600.
4. Оверлей обновляется автоматически; при использовании прямого API — опрашивайте каждые 5–10 сек.

## CORS

API отдаёт заголовки CORS для запросов с любого источника (публичный read-only API).
